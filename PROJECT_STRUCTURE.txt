Technology Stack
================

Frontend:    PyQt6 (cross-platform native UI)
Download:    yt-dlp (YouTube video downloading)
Processing:  FFmpeg (video/audio manipulation)
Logging:     Python's built-in logging [NEW]
Languages:   Python 3.10+
Deployment:  PyInstaller (standalone executables)

Logging System
==============

Location: ~/.media_downloader/logs/
Pattern:  TIMESTAMP - MODULE - LEVEL - MESSAGE
Levels:   DEBUG (detailed), INFO (normal), WARNING (alert), ERROR (failure)
Output:   Console (real-time) + File (permanent)
Coverage: Startup, downloads, processing, UI, errors, full tracebacks
